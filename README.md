# MetaEdit

A fast, standalone EXIF/IPTC metadata viewer and editor for macOS, built
for photojournalists and working photographers. Metadata is written to
industry standards (IPTC IIM + XMP dual-write, `.xmp` sidecars for RAW) so
files round-trip cleanly with Lightroom, Photo Mechanic, Bridge, and
Capture One.

Local-only: no network, no accounts, no AI, no telemetry.
